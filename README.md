# awesome_python3_webapp
Yes, that's right. This app is an awesome webapp build of python3.

### 需要的资源
```python
// 异步框架aiohttp
pip3 install aiohttp
// 前端模板引擎
pip3 install jinja2
// Mysql的Python异步驱动程序aiomysql
pip3 install aiomysql
```
### the tree for app
```
awesome-python3-webapp/  <-- 根目录
|
+- backup/               <-- 备份目录
|
+- conf/                 <-- 配置文件
|
+- dist/                 <-- 打包目录
|
+- www/                  <-- Web目录，存放.py文件
|  |
|  +- static/            <-- 存放静态文件
|  |
|  +- templates/         <-- 存放模板文件
|
+- ios/                  <-- 存放iOS App工程
|
+- LICENSE               <-- 代码LICENSE
```
